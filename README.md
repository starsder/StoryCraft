# StoryCraft

StoryCraft 是一个用于小说生成的 AI Agent。它旨在辅助创作者进行故事构思、角色设计、情节推演与文本生成，提供从灵感到大纲、从章节到完整小说的全链路支持。

## 项目状态

当前仓库处于基础骨架阶段，正在建立项目结构、分支模型与开发规范。核心功能与内容将逐步迭代添加。

## 分支模型

本项目采用简化的 Git 分支模型：

- `main`：稳定分支，仅用于发布已通过验证的版本。
- `develop`：开发分支，日常开发与功能集成在此进行。
- `feature/*`：功能分支，从 `develop` 切出，完成后合并回 `develop`。

请遵循上述分支策略提交代码，确保 `main` 始终保持可构建、可运行的稳定状态。
